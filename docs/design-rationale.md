# 設計根拠

## 結論

日本語の学術文書に、分野・大学・学会を横断する単一の版面指定はありません。一方で、長文本文の明朝体、A4縦・横書き、十分な余白、1字下げ、見出しと本文の明確な階層は、多くの資料で共通しています。

本テーマは次の2層で設計しています。

1. 日本語組版の共通層：明朝体、禁則を意識した改行、両端揃え、1字下げ、図表・脚注・見出しの分離
2. 用途別の表現層：読み物として自然な一般文書と、白黒で簡潔な学術文書

## 参照資料

### 日本語組版

- [W3C「日本語組版処理の要件」](https://www.w3.org/TR/jlreq/)
  - JIS X 4051を主な基礎とする日本語組版要件です。
  - 明朝体は日本語の長文本文で最も多く使われる書体と説明されています。
  - 本文は最終行を除き行頭・行末を揃えるのが原則です。
  - 新しい段落は原則として全角1字下げです。
  - 横組1段の版面例として、9pt・35字×28行・行間8ptが示されています。これは唯一の推奨値ではなく、版面設計の計算例です。

### 学術文書の例

| 資料 | 用紙・余白 | 本文 | 字数・行数 |
| --- | --- | --- | --- |
| [広島大学倫理学研究室「論文の執筆要領」](https://mlethica.hiroshima-u.ac.jp/education/writing-procedure/) | A4、四辺約30mm | 明朝体12pt以上 | 約35字×28行 |
| [筑波大学「学位論文（修士）の作成要領」](https://www.md.tsukuba.ac.jp/kango-kagaku/students/document/data/ex/sp_research12.pdf) | A4、四辺25mm | 明朝体10.5pt | 40字×30行、段落1字下げ |
| [山形大学「卒業論文・修士論文要旨集執筆要領」](https://www.tr.yamagata-u.ac.jp/~nobuyosi/%E5%8D%92%E8%AB%96%E7%99%BA%E8%A1%A8%E4%BC%9A%E8%A6%81%E6%97%A8%E5%9F%B7%E7%AD%86%E8%A6%81%E9%A0%98.pdf) | A4、四辺25mm | 明朝体10pt | 45〜50字を参考 |

資料間でも10〜12pt、35〜50字、25〜30mmと幅があります。このため、学術テーマは比較的よく見られる10.5pt・40字幅・25mmを初期値とし、CSS変数で変更できるようにしました。

### Typoraの仕様

- [Typora「About Themes」](https://support.typora.io/About-Themes/)
  - テーマフォルダ内のCSSファイルがテーマとして認識されます。
  - ファイル名は小文字の英字とハイフンを使います。
- [Typora「Add Custom CSS」](https://support.typora.io/Add-Custom-CSS/)
  - `{theme}.user.css` はテーマ本体より後に読み込まれ、利用者の上書きに使えます。
- [Typora「Export」](https://support.typora.io/Export/)
  - PDFの用紙、余白、テーマ、改ページ、ヘッダー／フッターはエクスポート設定から指定できます。
  - 印刷専用のCSSは `@media print` で定義できます。

### 同梱フォント

- [Noto Serif JP](https://notofonts.github.io/noto-docs/specimen/NotoSerifCJKjp/) は日本語に加えてラテン文字、ギリシャ文字、キリル文字などを収録するSerif書体です。
- [Notoの公式ライセンス説明](https://notofonts.github.io/noto-docs/website/use/)では、NotoフォントはOpen Font Licenseで提供され、条件に従った再配布が可能とされています。
- フォントファイルとOFL本文は [Google Fonts公式リポジトリ](https://github.com/google/fonts/tree/main/ofl/notoserifjp) から取得しています。
- 游明朝はWindowsで利用できますが、Microsoftの案内では再配布に別途ライセンス情報の確認が必要です。そのため同梱対象にはしていません。

## テーマへの反映

| 根拠 | 実装 |
| --- | --- |
| 長文本文では明朝体が一般的 | 游明朝、ヒラギノ明朝、Noto Serif CJK JP、源ノ明朝、MS明朝の順でフォールバック |
| 和欧文混植 | 学術テーマは同梱Noto Serif JPを全スクリプトへ使用し、字面とベースラインを統一 |
| 段落先頭は原則1字下げ | `--jp-paragraph-indent: 1em` |
| 行末を揃える | 本文段落に `text-align: justify` と `text-justify: inter-ideograph` |
| 行頭・行末禁則 | `line-break: strict` |
| A4、25〜30mmの余白が多い | `@page` をA4・25mmに設定し、30mm等へ変更できる手順を明記 |
| 見出し直後の天付きも許容される | 一般テーマのみ、見出し直後の段落を天付き |
| 見出しをゴシック体とする要領がある | 学術テーマの見出しはゴシック系フォールバック |
| 脚注は本文より小さく、罫線で区切る例が多い | 脚注を0.82em、上罫線付きで表示 |

## CSSで保証できないこと

- 指定フォントがない端末では、代替フォントの字幅と太さが変わります。
- `Japanese Academic` は同梱フォントを優先するため端末差を抑えられますが、`Japanese Print` はOSフォントの差が残ります。
- `40em` は全角40字相当の幅ですが、約物、欧文、数式、プロポーショナル字形により実際の文字数は変化します。
- TyporaのPDF設定が `@page` より優先される場合があります。
- CSS Paged Mediaだけでは、提出要領ごとに異なる柱、ノンブル、表紙、奇偶ページ、厳密な行グリッドを完全には統一できません。
- 縦組はTyporaの公式資料でも制約が示されているため、本テーマの対象外です。

提出前には、PDFの用紙サイズ、余白、フォント埋め込み、改ページ、図表番号、参考文献形式を提出先の要領に照らして確認してください。
