# Noto Serif JP

`NotoSerifJP-Variable.ttf` is the unmodified Noto Serif JP variable font from the official [Google Fonts repository](https://github.com/google/fonts/tree/main/ofl/notoserifjp).

- Family: Noto Serif JP
- Weight axis: 200–900
- Upstream Google Fonts commit: `8a7c74854f766ae441c7584925cc0ec626fc5aa6`
- SHA-256: `2fd527ba12b6a44ec30d796d633360da0aeba6c5d4af1304ce12bb4dc15a7dfc`
- License: SIL Open Font License 1.1 (`OFL.txt`)

The font file has only been renamed for a filesystem-safe, stable theme resource path. Its contents are unchanged.
